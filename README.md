# interittus13.in — Personal Blog Application

> A personal blog exploring ideas, projects, and perspectives in tech, design, and beyond.

## 🚀 Features

- ✍️ Markdown-based content
- 🌐 SEO & PWA ready (Next.js + next-seo + next-pwa)
- 🎨 Syntax highlighting with Prism
- 🌙 Dark mode support via `next-themes`
- 📈 Google Analytics & Baidu Analytics tracking
- ⚡ Optimized static generation with ISR (Incremental Static Regeneration)
